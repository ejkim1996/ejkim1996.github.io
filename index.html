TUTORING CENTER SCHEDULE CREATOR

Google Cloud Firestore | Google Calendar API | express.js | node.js | handlebars.js | React.js

Every semester, the directors of the University Learning Center at NYU need to manually update the weekly schedule provided on the website. This process takes up to 3 days since the tutoring center has around 70 tutors, each teaching 10-20 subjects. This web app reduces the time it takes to 30 minutes by first grabbing shift schedules of every tutor using the Google Calendar API, and then using an interval merging algorithm to provide correct times for every subject.
